# 709. To Lower Case
###### Easy

Given a string s consisting of words and spaces, return the string after replacing every uppercase letter with the same lowercase letter.

#### Example 1:

<b>Input:</b> s = "Hello"

<b>Output:</b> "hello"


#### Example 2:

<b>Input:</b> s = "here"

<b>Output:</b> "here"


#### Example 3:

<b>Input:</b> s = "LOVELY"

<b>Output:</b> "lovely"
 

#### Constraints:
* 1 <= s.length <= 100
* s consists of printable ASCII characters.
